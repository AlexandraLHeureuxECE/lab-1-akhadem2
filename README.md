# Tic-Tac-Toe (Browser Game)

A simple, browser-based Tic-Tac-Toe game built with plain **HTML, CSS, and JavaScript**.  
The game supports two players on the same device, provides clear turn feedback, detects wins and draws, and allows the game to be restarted without refreshing the page.

This project was created as a lightweight front-end exercise and is suitable for deployment on GitHub Pages.
